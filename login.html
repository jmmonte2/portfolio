Login application

Motivation
The reason I decided to work on this project was to gain an understanding on how login systems are implemented.

Key Takeaways
User Authentication
Each time a user attempts to login a query is being executed to determine if the credentials passed were accurate.

Security
Made use of hashing when storing passwords to protect sensitive information.

Virtual Environment
Learned that Python projects need a virtual environment to store all neccesary dependencies, which is also very useful when deploying an application.

HTTP Requests
This project made use of POST and GET request. POST was utilized each time a user signed in or created an account, since they were sending information to the server.
